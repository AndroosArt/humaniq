# New Client Onboarding Checklist

## Internal kickoff readiness
- [ ] Correct engagement path is confirmed
- [ ] Final approved scope is captured
- [ ] Payment or deposit is confirmed
- [ ] Owner is assigned
- [ ] Welcome email is ready
- [ ] Client intake request is ready
- [ ] First-week target outcome is clear

## Client info and assets needed
- [ ] Primary point of contact
- [ ] Decision-maker or stakeholder list if relevant
- [ ] Business overview or context needed for the engagement
- [ ] Current offer or sales material in scope
- [ ] Existing process notes, documents, or screenshots
- [ ] Any relevant prior recommendations or internal notes

## Access requests
- [ ] Website or landing page access if needed
- [ ] CRM or pipeline access if needed
- [ ] Scheduling tool access if needed
- [ ] Analytics or reporting access if needed
- [ ] Shared folder or document access if needed

Request only what is necessary for the first phase.

## Implementation prerequisites
- [ ] Scope is bounded enough to begin
- [ ] Key dependencies are known
- [ ] Missing inputs are identified
- [ ] Timeline for first deliverable is realistic
- [ ] No unpaid work is being started by accident

## Scope confirmation
- [ ] What is included is clear
- [ ] What is not included is clear
- [ ] First milestone is defined
- [ ] Practical AI Blueprint role is clear if relevant

## First-week actions
- [ ] Send welcome email
- [ ] Send client intake request
- [ ] Confirm receipt of onboarding materials
- [ ] Review client inputs as they arrive
- [ ] Send first action summary or milestone note
- [ ] Schedule optional kickoff only if needed
