# Human IQ

Human IQ is a founder-led consulting launch project focused on a clear front-door offer and a practical follow-on diagnostic.

## Offer Structure
+Front-door offer: Growth & Conversion Review
+Downstream diagnostic/deliverable: Practical AI Blueprint

## What This Repo Is For
+Organizing launch-prep assets and documentation
+Tracking the current sprint and launch checklist
+Giving Codex a clean, safe workspace for execution
+Creating a simple foundation that can grow later without rework

## Current Status
+This repository is in launch-prep.
+The landing page direction and intake materials already exist conceptually, but they are not yet present in this repo.
+GitHub is not connected yet.

## Recommended Structure
```text
.
├── .codex/
│   └── config.toml
├── docs/
│   ├── current-sprint.md
│   └── launch-checklist.md
├── AGENTS.md
├── README.md
└── .gitignore
```

Add more folders only when they are needed, for example:
+`site/` for landing page code
+`assets/` for approved visuals and brand files
+`docs/intake/` for intake forms, scripts, and qualification material

## Minimum Workflow
1. Plan the next small batch of work in ChatGPT.
2. Capture the active scope in `docs/current-sprint.md`.
3. Use Codex to implement or edit files in this repo.
4. Commit meaningful milestones to GitHub.

## Notes
+If existing landing page files or intake assets live elsewhere on your machine, they should be intentionally copied into this repo instead of being loosely referenced.
