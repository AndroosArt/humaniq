# Founder Review Checklist

Use this when a new Growth & Conversion Review submission arrives.

1. Open the latest Formspree submission and read `ops_founder_review_snapshot` first.
2. Confirm the basics are present: business name, contact name, email, and website.
3. Review `lead_tier`, `primary_problem_category`, `engagement_recommendation`, `optional_call_recommended`, and `response_sla`.
4. Copy `ops_tracker_row_csv` into the tracker, or paste the values into the matching columns in `docs/ops/lead-tracker.csv`.
5. Confirm the lead has exactly one primary path before anything is sent.
6. Set `Status`, `Owner`, and `Next action` if they need to change after review.
7. Send the first async recommendation or proposal using `ops_follow_up_subject` and `ops_follow_up_body`, adjusting only if a more tailored note is clearly better.
8. Use a call only if it would help clarify scope, answer buyer questions, or support onboarding.
9. Add any founder-only context to `Notes`, then make sure nothing strong is left waiting past the SLA.
