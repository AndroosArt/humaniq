# Founder Review Checklist

Use this when a new Growth & Conversion Review submission arrives.

1. Open the latest Formspree submission and read `ops_founder_review_snapshot` first.
2. Confirm the basics are present: business name, contact name, email, and website.
3. Review `lead_tier`, `primary_problem_category`, `engagement_recommendation`, and `response_sla`.
4. Copy `ops_tracker_row_csv` into the tracker, or paste the values into the matching columns in `docs/ops/lead-tracker.csv`.
5. Set `Status`, `Owner`, and `Next action` if they need to change after review.
6. Send the first follow-up using `ops_follow_up_subject` and `ops_follow_up_body`, adjusting only if a more tailored note is clearly better.
7. If the submission looks strong and time-sensitive, invite the Practical AI Blueprint conversation the same business day.
8. Add any founder-only context to `Notes`, then make sure nothing strong is left waiting past the SLA.
